# Kiwix 离线资料

https://kiwix.org/en/   
- kiwix, 支持离线阅读, 可以下维基百科 (Wikipedia) 的离线版（.zim 格式）
- 推荐下载. (Language: 中文, English)
  - 对大文件可以直接通过 [官网ZIM链接](https://download.kiwix.org/zim/wikipedia/) 下载
  - 维基百科, 中英文完整版    
    - 医学维基百科. 450MB. 便携设备应急用.
    - 地理维基百科. 300MB. 便携设备应急用.
    - WikiMed Medical Encyclopedia. 2GB. 英文医学维基百科.
    - Geography by Wikipedia. 1.3GB. 英文地理维基百科.
    - wikisource, 中文完整版. 包含大量古籍.
    - wikisource, 英文完整版 包含已出版作品的原文
  - wikibooks, 中英文完整版. 现代教程为主.
  - wikiHow, 中英文完整版. How-to, 怎么做的简易指南.
    - 官网已经下载不到.
    - 可以从 [kiwix zim 镜像备份](https://cdimage.debian.org/mirror/kiwix.org/archive/zim/wikihow/) 下载
  - 古腾堡图书馆项目, Project Gutenberg Library 内所有的中文书. 古籍为主.
  - ~~Project Gutenberg Library~~  七万多的电子书, 以文学为主. 内含传统农业和食品保存

# 如何利用这些资料

- 在电脑端和手机端都装好 kiwix, 
  - 将下载好ZIM资料放进kiwix设置中的目录即可自动识别.
  - **kiwix 以自己查询阅读为主.**
  - 手机端受限于容量, 可以只放核心ZIM文件. 
  - 外置一个Type C U盘, 存放ZIM大文件, 让手机可访问.
  - 手机必须考虑电量问题, 需要为其准备太阳能充电板.
- 大量的PDF文件, 两种用法. 
  - 一种为明确需求后, 自己查阅学习.
  - 另一种为利用本地AI, 建立知识库. 可参考 [本地AI配置教程](https://draapho.github.io/2026/02/25/2607-LocalAI/)
- 仅仅下载资料是毫无用处的. 但立刻开始练习掌握也不现实. 重点是有所准备
  - 总体的准备原则是在一段时间内, 尽可能的自给自足.
    - 可假设所有的公共服务, 即水电网, 购物都无法稳定的提供.
  - 此种情况下, 尽可能根据现实情况和自身条件, 最大限度的实现一定范围内的自给自足.
  - **心理准备是第一层, 物质准备是第二层, 身体锻炼适应是第三层.**
- **有稳定居所, 且居所安全的, 重点关注食物和能源的自给自足.**
  - 能源直接安装太阳能板+电池储能. 有条件的购买纯电车及必要的电动工具.
  - 食物参考学习家庭农场的运营. 另要保证水源.
  - 提前将必要的软件和资料下载保持并备份. 
  - 可考虑部署本地AI, 知识的学习和获取将非常便利.
- 户外生存模式下, 充分准备和无准备有非常大的差距.
  - **书籍里提到的应急物资和工具, 必须要提前整理打包好.**
  - **户外环境一般较为恶劣和不可控. 不要过度依赖电子产品.**
  - **第一层知识储备: 准备太阳能板, 线材, 手机. 资料存放在手机和备用U盘.**
  - **第二层知识储备: 准备精心挑选的一本纸质书籍, 手摇式手电筒, 氚气管氚灯.**
