# 前言

个人认为, 八字四柱是很好的定性工具, 而非预测工具.
它的预测只可能是概率性的, 某些特定的命盘结构, 确定性较大, 普通命盘结构, 能有个50%的预测准确性就很好了. 如果要用八字预测, 必须配合其它辨人,预测技术, 综合各种信息后, 才能进一步提高预测准确度. 另外, 心有所感, 专事专问的情况下, 也可通过八字的大运流年来判断, 正确率会有所提升.

但**八字四柱, 非常适用于判断自身的特质, 在性格特质, 学习方式的偏好, 人生职业规划, 择偶性格互补相合**等方面, 有着极其巨大的应用潜力. 远比星座, MBTI的准确度, 便利度要高. 而且和MBTI, 心理测试并不冲突, 可相互补充.

## 十神
**十神**, 是四柱的专用概念, 也是四柱最重要, 最核心的概念. 以**日主**为中心, 即日柱的天干, 代表自己. 基于五行生克和阴阳关系推导而来.

| 十神 | 与日主的关系    | 简称及作用   | 特性和偏好 | 抽象意象      | 人事意象                    |
| ---- | --------------- | ------------ | ---------------- | ---------------------------- | ---------------------------- |
| 比肩(比) | 同五行,同阴阳   | **比劫** 旺 100% | *合作* | 自我,竞争,助力 | 自尊,同辈,兄弟朋友 |
| 劫财(劫) | 同五行,异阴阳   | *资源获取* | *竞争* | 争夺,变动,外助 | 竞争,合作,帮扶       |
| 正印(印) | 生日主,异阴阳 | **印星/印枭** 相 90% | *外部认可* | 学识,安全,仁慈 | 学历,贵人,母亲 |
| 偏印(枭) | 生日主,同阴阳 | *性格底色* | *自我内求* | 偏门,孤傲,洞察 | 玄学,绝活,继母 |
| 食神(食) | 日主生,同阴阳 | **食伤** 休 30% | *物质/享乐* | 才华,享受,温和 | 技艺,福寿,子女(女命) |
| 伤官(伤) | 日主生,异阴阳 | *欲望层面* | *精神/冒险* | 创意,叛逆,表达 | 露才,创新,子女(男命) |
| 正财(财) | 日主克,异阴阳 | **财星** 囚 50% | *稳定节俭* | 稳定,务实,节制 | 工资,勤俭,责任,妻子    |
| 偏财(才) | 日主克,同阴阳 | *财源特性* | *波动大方* | 意外,投机,慷慨 | 商业,社交,情人,父亲 |
| 正官(官) | 克日主,异阴阳 | **官杀** 死 10% | *外部规训* | 规则,责任,名誉 | 仕途,自律,丈夫 |
| 七杀(杀) | 克日主,同阴阳 | *压力来源* | *自我要求* | 压力,挑战,魄力 | 武职,危机,小人 |



## 其它专用术语

初学可忽略, 需要时再学习相关术语.
- 地支**透干**, 所有天干对指定地支藏干(本气, 中气, 余气)是否五行比和. 最强为主, 其它最多酌情+30%.
  - 天干与该地支藏干五行比和, 即透干, 对指定地支有助力. 
  - 天干与该地支藏干, 有比和即为**透干**. 
    - 距离近且与本气比和, 该地支有额外助力. 120%
    - 距离远, 还是中气/余气, 酌情减弱. 70-90%
  - 所有天干, 与该地支藏干无比和, 为**无透**. 
    - 该地支大为减弱. 60%
- 天干**通根**, 所有地支藏干(本气, 中气, 余气)对指定天干是否五行比和. 最强为主, 其它最多酌情+30%.
  - 地支藏干与指定天干五行比和, 即通根, 对指定天干有助
  - 地支藏干的本气100%, 中气50%, 余气10%-30%(可暂忽略).
  - 地支与指定天干的距离, 相邻100%, 相隔80%, 更远50%, 再远则忽略.
    - 中间必须没有冲合阻断.否则作用力进一步下降.
  - 所有地支藏干, 与该天干无比和, 为**无根**.
    - 该天干大为减弱. 60%
- **盖头**: 必须同柱, 天干克地支. 地支气弱. 50%
- **截脚**: 必须同柱, 地支克天干. 天干气弱. 50%
- **星运**: 对日干而言, 取出四柱地支的十二长生. 星运是外在, 时空环境.
- **自坐**: 对天干而言, 基于同柱地支的十二长生. 自坐是内在, 核心气质.
- **专旺**: 针对日主而言. 日主得令+得地+得势, 全局比劫印星极旺,无强力的财官食伤, 即专旺.
  
  - 专旺格"喜顺不喜逆"，大运走印比食伤则吉，走财官则凶。
- **从格**. 所谓从格, 即日主弱(从强格除外), 某五行独旺.
  - **从财**格: 财星旺(我克者), 喜财星/食伤,忌印星或比劫
  - **从杀**格: 官杀旺(克我者), 喜财星/官杀,忌食伤(会克官杀)
  - **从儿**格: 食伤旺(我生者), 喜食伤/财星, 忌印星(会克食伤)
  - **从强**格: 比劫旺/印星旺(帮我者), 喜印星比劫, 忌财星(耗日主)或官杀(克日主)
  - 独旺的要求, 概括来说, 就是对该五行, 得令+得地+得势(见"日主强弱"). 即:
    - 得令:月令比和或生该五行;
    - 得地:地支有该五行的本气根,或成三合三会局;
    - 得势:天干比和或生该五行,且没有强克.
  - 除从强格外, 另三格如果四支的中气余气中有比劫或印星(日主有微根/微助), 判定为**假从**.
  - 相关从格, 如果四支的中气余气中有忌神, 判定为**假从**. 如果是天干或本气有忌神, 直接降为普通格局.
  - **假从**依然是从格, 但有杂质. 更易被破格. 如果大运流年能压制日主的微根(冲)微助(克), 则顺.
  - **破格**: 如果是大运/流年里出现了从格的忌神, 破坏了原来的格局, 即破格, 表示运势受阻.
- **调候**: 调节八字寒暖燥湿. 使其阴阳平衡. 
  - 确定是否存在失调. 失调才需要调候.
  - 四柱的调节效果为, 月干=时干=日支本气>年干>其它地支藏干本气>地支中气余气(仅透干情况)
  - 先看月支, 即出生月份. 注意: 南半球不用颠倒, 是固定参数!
    - 冬月(亥子丑), 极寒, 水旺. 需火(丙丁)来调节
    - 夏月(巳午未), 极热, 火旺. 需水(壬癸)来调节
  - 再统计四柱五行. 两个相生五行>=4, 而能克它的五行为0, 则失衡了. 
    - 木火多而无金水, 燥热焦枯. 需水调节.
    - 火土多而无水木, 炎燥板结. 需水木调节.
    - 土金多而无木火. 寒湿凝滞. 需火木调节.
    - 金水多而无火土. 寒湿冰冷. 需火土调节.
    - 水木多而无火土, 寒湿泛滥. 需火土调节.
  - 例外情况: 
    - 日主专旺格, 喜顺不喜逆, 不求平衡.
    - 从格若顺应月令气候(如夏月从火格,冬月从水格), 无需调候.
- 命局**流通**: 指命局中五行之气**无严重阻滞,最终能归于日主或用神**的状态
  - 简单理解的话, 为相邻的四柱, 有生扶为顺畅. 有冲克合绊, 有强的刑害破为阻塞.
  - 需要注意气场强弱, 譬如生者不能过弱, 受者不能过旺. 譬如克者过弱, 是可以被反克的(即相侮).

## 二. 日主强弱

从三个维度综合判断日主气场的强弱, 数值仅供参考.
- 2.1 基于月支, 即出生的月份, 判断日主, 即自己的气场强弱.
  - 初学时, 固定权重50%. 高阶需灵活调整
  - 传统术语"月令", **得令/失令. 代表天时**, 宏观大环境.
  - 月支本气**比劫**, 日主得月令同助, **得令**. 数字化: 100%
  - 月支本气**印星**, 日主得月令生扶, **得令**. 数字化: 90%
  - 月支本气**食伤**, 日主需要生月令, **失令**. 数字化:  30%
  - 月支本气**财星**, 日主需要克月令, **失令**. 数字化:  40%
  - 月支本气**官杀**, 日主被月令克制, **失令**. 数字化: 10%
  - 一般只看本气就足够了. 月支中余气最多对本气影响(增强或减弱)15%以内.
- 2.2 四柱地支的藏干(本中余), 是否比和日主. 
  - 初学时, 可固定权重30%. 高阶需灵活调整
  - 传统术语**得地. 代表地利**, 表示地基,根系,个人基础.
  - **得地**: 主取最强气场的地支, 其它可做辅助(譬如最多30%的助力)
    - 在地支藏干中, 看是否有 **比劫** 即可.
    - 距离比重, 月支,日支: 100%; 年支,时支:90%. 
    - 地支藏干与日主比和的占比: 本气100%; 中气50%; 余气10%
  - **失地**: 地支藏干, 对日主无任何生扶作用. 则根基不稳.0%
- 2.3 查看三柱的天干十神, 确定整体天干对日主的气场强度.
  - 初学时, 可固定权重20%. 高阶需灵活调整
  - 传统术语**得势  代表人和**. 是否有帮衬,助力. 综合累加.
  - 助力方: **比劫, 印星**
    - 正印(85%) 偏印(80%) 比肩(100%) 劫财(100%)
  - 耗泄方: **食伤, 财星, 官杀**
    - 食神(-60%) 伤官(-70%) 正财(-50%) 偏财(-55%) 正官(-90%) 七杀(-100%)
  - 日干距离: 离日干进, 则作用强, 离日干远, 则作用弱.
    - 紧贴:月干,时干,日支: 100%.
    - 斜隔:年干,月支,时支 80%.
    - 远隔:年支,50%



初学时, 按上面的简化步骤进行粗略的定性判断就可以了. 
更深入全面的情况, 还要判断对日主产生作用的各个干支, 自身受外界影响, 而产生的强弱变化.
- 2.1 得令, 对月令(月支)强弱的补充判断.
  - 月支有三会三合(150%), 大大加强
  - 月支与它支有冲(50%)刑(70%)害(80%), 减弱
  - 月支旬空, 气虚. (50%)
  - 月支无透, 弱. (60%)
- 2.2 得地, 对剩余三(年日时)地支强弱的补充判断.
  - 该地支被**盖头**(被同柱天干克), 无气. (50%)
  - 该地支旬空, 气虚. (60%)
  - 该地支与它支有冲(50%)刑(70%)害(80%), 减弱. 可忽略地支相互间的五行生克. 
  - 该地支**透干**, 但只是中余气远干, 减弱(70-90%). 如果**无透**, 弱. (60%)
- 2.3 得势, 对三(年月时)天干强弱的补充判断.
  - 该天干被**截脚**(被同柱地支克), 无气. (50%)
  - 该天干与邻近天干, ~~被生(110%)~~, 被克(60%), 合绊(50%). 相隔天干无作用关系.
  - 该天干**通根**, 只是通远支中余气, 减弱(70-90%). 如果**无根**, 气弱(60%)
  - 该天干, **自坐**墓绝. (50%). 但有出墓和绝处逢生(三合/合/冲墓/绝逢生), 则转强.
- 2.4 其它情况说明
  - 三会局, 四柱地支见三会, 成局. 代表季节专气, 力量最强. 150%. 注意过旺失衡.
  - 三合局, 四柱地支见三合. 如果见二合则为半合, 需透干或月支生扶. 130%. 注意过旺失衡.
  - 合化条件, 气场强且稳定. 1.月支生扶; 2.无强冲破害; 3.天干比和合化五行(引化)且不克合化五行. 
    - 合化条件非常上述1,2,3必须同时满足, 否则就是合绊. 
    - 合化成功后, 需要使用合化五行重新计算日主强弱.
    - 透干引动的强度, 影响合化后的气场强度. 无透干:化气不纯:60%



注意, 以上百份比只是参考, 并无合理性验证. 自动化编程还需考虑如下情况对数值的影响..
- 特殊格局必须前置判断. 以下情况会彻底推翻常规强弱逻辑. 需在量化前单独拦截:
  - 从格（从强,从财,从杀,从儿）
  - 日主专旺（曲直,炎上,稼穑,从革,润下）
  - 合化成功, 需要用合化五行重新计算强弱.
- 强弱只是“量”，调候（寒暖燥湿）与生克路线（气机是否流通）决定“质”。例如:
  - 冬水日主，即使得令有根，若无火调候，仍为“寒凝无力”。
  - 天干官印相生,地支财官印连续相生，即使日主偏弱，亦能任财官。





## 三. 定格局,取用神

取用神的核心理念是: **阴阳平衡**. 直觉上可能绝对弱不好, 但过旺同样是不好的. 取用神的时候, 过弱就要取能帮扶的日主的十神, 过强就要取能压制平衡日主的十神. 但特殊命格另论.

| 术语     | 原则                                  | 作用和说明                                                   | 选用方法                                                     |
| -------- | ------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **用神** | 先定五行,再定阴阳<br>最后定得气的干支 | 命局中对日主起到平衡作用的十神<br>作为平衡命局的"药", 需气场适中, 不宜过多过强 | 扶抑: 平衡日主强弱<br>调候: 调和寒暖燥湿<br>通关: 化解五行战克 |
|          | **扶抑** (日主旺衰)                   | 适用于日主旺衰明显的普通命局.<br>强则抑, 弱则扶, 中则和. 注意用药强度 | 日主强: 取官杀, 食伤, 财星<br>日主弱: 取印星, 比劫<br>日主和: 确保命局流通, 或取调候 |
|          | **调候** (季节失衡)                   | 月支夏季燥热或冬季湿寒, 气候失衡<br>或四柱五行严重偏科, 则调候优先.<br>总原则, 阴阳平衡, 燥湿平衡, 寒热平衡 | 月支春: 丙火(泄木), 葵水(润木)<br>月支夏: 壬水优先, 庚金辰丑<br>月支秋: 丁火(炼金), 壬水(润金)<br>月支冬: 丙火优先, 戊土丁火 |
|          | **通关** (两神相战)                   | 两神等势相战(如金木, 水火), 需通关调和                       | 取通关五行. 金木战取水,<br>木土战取火, 土水战取金,<br>水火战取木, 火金战取土 |
|          | **专旺/从格**                         | 顺势用神,不可逆选尝试平衡                                    | 用神选顺喜之神即可.                                          |
|          | **化气格**                            | 日主有合且合化成功(条件极其苛刻)                             | 取合化五行为用神                                             |
| **喜神** |                                       | 辅助用神达成平衡的五行<br>*不以五行生克为标准*               | 生扶日主,生用神<br>克忌神,使用神通关                         |
| **忌神** |                                       | 破坏命局平衡,往往意味着引发灾咎.<br>*不以五行生克为标准*     | 克用神,严重耗泄用神<br>克日主,破坏通关                       |
| **仇神** |                                       | 助长忌神,削弱喜神,不利命局平衡<br>*不以五行生克为标准*       | 生忌神,克喜神                                                |



## 四. 初步推算

动态观察用神的位置, 以及动态作用路径.

| 检视维度       | 核心问题                                                     | 实战要点                                                     |
