# 近期卦象的数字分析

## 第四一象　甲辰 ䷝ 離下離上 火

颂曰: `帽儿须戴血无头 手弄乾坤何日休 九十九年成大錯　稱王只合在秦州`
九十九年, 三种解法.
- 直指 99年
- 9+10+9=28年
- 90+19 = 109年
- 另有, 万年歌, 更得相安一百九

时间基点: 
- 前文 `帽儿须戴血无头` 是文革. `手弄乾坤何日休` 反手为毛, 大跃进和文革.
- 故首选建国1949
- 次选中共成立1921.

有如下组合:
- 1949+28 = 1977. 
  - 1976毛去世. 1977年恢复高考, 1978年改革开放.
  - 可理解为建国后28年里毛犯的大错.
  - 较为勉强, 不应后句 `稱王只合在秦州`
- 1949+99 = 2048 / 1949+109 = 2058
  - 这两个时间点为主选. 符合四一象总起PRC, 
  - 前两句为建国后的运动, 后两句为失去权力的时间和归宿.
  - 另可参考梅花诗其八: `如棋世事局初残，共济和衷却大难。豹死犹留皮一袭，最佳秋色在长安。`
  - 不做猜测, 静观历史.  
- 1921+28 = 1949
  - 时间很匹配, 但不符合成大错. 也不应前面两句的时间点.
-  1921+99=2020
  - 武汉爆发新冠病毒.  
  - 仅作备选, 可以作为由盛转衰的转折点. 
- 1921+109 = 2030
  - 不做猜测, 静观历史.  



## 第四三象　丙午 ䷱ 巽下離上 鼎

颂曰:  `黑兔走入青龙穴 欲尽不尽不可说 惟有外边根树上 三十年中子孙结`

三十年中, 三种解法
- 30/2=15年
  - 参考`十二月中气不和`, 为十二月取半, 农历六月.
- 直指 30年
- 3+10=13年

时间基点: 
- 可能是黑兔青龙. 即2023到2024年
- 亦有可能是《马前课》,第十一课: `晨雞一聲 其道大衰`
- 最近的是己酉年2029年, 为土鸡. 晨同"辰", 五行为土.

有如下组合:
- 2023, 2024+30=2053, 2054年. 
- 2023, 2024+15=2038, 2039年.
- 2023, 2024+13=2036, 2037年.
- 2029+30=2058年.
- 2029+15=2044年.
- 2029+13=2042年.

分析:
- 此象的图和谶都是两岸关系. 因而认为它的颂, 也是在着重写两岸关系. 是国共分命运分述后的总结.
- `黑兔走入青龙穴 欲尽不尽不可说` : 
  - 指的是2023, 2024之后, 中国台湾的关系`欲尽不尽不可说`.
  - 也就是2024年"二合一选举"后, 民进党第三度执政, 台独者赖清德当选. 两岸关系紧张, 似有随时擦仓走火之顾虑.
  - 但 `欲尽不尽不可说` 和 `终克定` 说明了最终没有发生武统.
- `唯有外边根树上 三十年中子孙结`: 
  - `外边根树`为台湾.  
  - `子孙结`为福禄, 即安稳渡过, 没有灾祸. 
  - `唯有`, 即只有台湾为福禄. 暗示大陆端动乱.
  - `三十年中`, 最有可能的时间是2029-2058年.
    - 从明显的动乱年份算起. 有预言暗示的是己酉年2029年
  - 《推背图》,第四十一象: `九十九年成大错 称王只合在秦州`
    - 九十九为国祚. 可解为: 1949+90+19=2058
  - 《万年歌》, `两分疆界各保守。更得相安一百九。`
    - 国民党败退台湾是1949年. 可解为: 1949+109=2058
