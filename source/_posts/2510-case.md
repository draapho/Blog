# 基于时间信息, 猜测物品

- 打一个能用杯子遮盖起来的物品. Youtube, 总裁聊聊, 搜索射覆. 
  - 网址: https://www.youtube.com/watch?v=OAzkZTWoPFk
- 时间起卦法: 2025年9月11日15点49分
  - 乙巳年八月十六日申时
  - 巳6年 8月 16日 申9时 
  - 上卦:  (6+8+16)%8=6坎
  - 下卦:  (6+8+16+9)%8=7艮
  - 动爻:  (6+8+16+9)%6=3
- 本卦: 水山蹇, 动爻九三, 变卦: 水地比
- 解卦: 有疏通之用的液体物品
  - 水不变, 该物品多半内为液体, 为体(虽有背于下体上用, 但更符合实际情况)
  - 水山蹇, 堵塞, 阻碍, 不通.
  - 九三: 来反, 以治其内, 为得自全.
  - 水地比: 下顺从. 通畅了.
  - 高度怀疑为药用物品. 主搜山对应的人体器官 
    - 艮山: 主要对应人体的手指、骨骼、鼻、背部.
  - 该物品功用: 通鼻, 活血化瘀(背部)
  - 故可猜测: 鼻喷雾剂, 鼻吸入剂, 红花油.
- 答案是鼻烟壶, 可以应验.



# 基于开战时间, 评估俄乌战事

- 时间起卦法: 2022年2月24日3点40分.
  - 壬寅年正月二十四日寅时
  - 上卦:  (3+1+24)%8=4震
  - 下卦:  (3+1+24+3)%8=7艮
  - 动爻:  (3+1+24+3)%6=1
- 本卦: 雷山小过. 动爻初六. 变卦:雷火丰
- 解卦: 起步即凶, 轻动招祸, 先盛后竭
  - 震雷常表战争.
  - 雷山小过: 可小事, 不可大事. 宜下, 不宜上. 此刻动武, 大凶!!!
  - 初六: 宜按兵不动, 但俄罗斯动兵了, 必出师不利, 初战即受挫.
  - 雷火丰:  双方大动干戈. 先躁进，后困顿；外盛内虚，得不偿失.
- 可以说是完全应验.




# 2025年12月8日, 问2026是否发生全球性金融危机
- 略筮法占卜说明略. 问2026是否发生全球性金融危机
  - 得: 上卦8坤, 下卦1乾, 动爻2
- 本卦:地天泰, 动爻九二, 变卦:地火明夷
- 解卦: **上半年还欣欣向荣, 第三季度突然的经济崩溃!!!**
  - 坤卦可直接对应全球.
  - 地天泰: 可以理解为过去的经济情况. 泰极并非好事. 重点看动爻了.
  - 九二: 要包容广大, 不结党营私, 舍利守中, 才是上策, 能维持泰.
    - 强调了表面繁荣下要有涉险意识. 
    - “包荒”暗示系统内已藏隐患(如泡沫,债务,结构性失衡),但因包容性强,尚未爆发
    - “朋亡”警示:若因利益集团裹挟((如金融寡头,政策短视), 则中道难守
    - 措施合理, 则危机可避免,但需高度警觉与改革.
  - 应比:
    - 九二是处理潜在危机的最后机会(2026年三四月)
    - 九三为极, 六四为变. 四爻可对应七八月.
  - 地火明夷: 表面平静,内里危机; 贤者受伤,小人得势; 经济隐忧爆发
    - 运当大难.  后续必会有突然性的黑天鹅事件, 市场信心崩溃, 危机爆发.
    - 非局部危机, 而是系统性, 全局性的“光明被掩”.
    - 大概率在夏秋之交(7月到9月)
  - 整体卦象: 泰极而"明夷", 不是缓慢衰退, 而是突然陷入黑暗
  - 人和的问题: 世界明显做不到包容广大, 舍利守中. 而在各自为战, 结党营私!!!
- 2026年1月16日, 综合问卦黄金后的更新: **第一季度交易市场上涨甚至新高, 5月份第一波衰退, 到时是购入黄金的最后机会. 第三季度, 倾向于9月份全球确认金融危机到来, 黄金单边上涨**
- 是否应验: 静待时日.




# 2025年12月10日, 问未来五年, 是否会发生世界大战.

- 略筮法: 上卦5巽, 下卦4震, 动爻6
- 本卦:风雷益, 动爻上九, 变卦:水雷屯
- 解卦: **极有可能发生大战, 大难. 战后艰难, 有新政权出现.**
  - 震卦可表战争
  - 风雷益: 迅雷烈风, 改旧换新, 在此时也. 若能损上益下, 自上下下, 其道大光, 善.
  - 上九: 莫益之, 或击之. 立心勿恒, 凶. 太过则变, 变则不为益而为击矣. 易有意外之祸
    - 现实情况是底层民生越发艰难, 穷者更穷, 富者更富, 社会越发动荡.
    - 美国优先, 不断退出国际组织, 政治经济货币信誉都严重受损. 
    - 从民从国看, 都符合责下而益上, 立心不恒, 惟利是图.
    - 综合判定: 大凶. 必变而击. 容易发生意外之祸.
  - 水雷屯. 刚柔始交而难生. 处屯难之会, 当动性忍心, 坚贞自持.
    - 上坎水, 为险、血、陷、盗.
    - 《象》曰：“云雷，屯；君子以经纶。”: 需出圣人, 以极大智慧织理乱局，非自然平复。
    - 屯卦“利建侯”：意味着战后将出现新权力中心（如区域联盟、新霸主），但过程充满险阻。
- 是否应验: 静待时日.



# 20251222: 知乎MRWEEKEND的危机预测(经济,自然灾害,战争,核战, 战后困顿), 是否基本正确(会发生, 允许日期误差)?

- 略筮法: 上卦1乾, 下卦3离, 动爻5
- 本卦: 天火同人, 动爻九五, 变卦:离为火
- 解卦: **所言具公共合理性，随着事态的发展, 核心判断将被事实印证, 获得群体认同**
  - 本卦: 天火同人. 同人于野，亨，利涉大川. 无所容其私心, 斯物无不应, 人无不助. 君子正也
    - 乾（天、君、公信力）上，离（火、明、舆情、网络）下
    - 同人卦象征公开讨论、群体共识、信息在开阔之地（“野”）传播
    - 其言得众信，因无私而通。
  - 动爻: 九五. 同人，先号咷而后笑，大师克相遇。
    - 初期遭质疑、恐慌（号咷），终因事实验证(相遇) 而众人信服（笑）. 
  - 变卦: 离为火. 上下皆离，重明继照，光明盛大，亦有附丽、依附、信息过载之象
    - 该预测将持续引发关注，成为公共议题，被反复审视
    - 离卦问灾, 主火灾, 兵戈, 热疫! 发生火山爆发, 疫情, 金融泡沫破裂, 战争, 都可视为离象之应.
  - AI的提醒: 可信 ≠ 必然发生, “事实印证” ≠ “危机全面爆发”
    - 指其分析逻辑自洽、动机公允、具预警价值——此卦支持
    - “必然发生”：指预言的事件必定兑现——此卦未支持
    - “印证”的对象，是“风险存在”或“系统脆弱性”，而非“预言的每个灾难场景必定上演”
- 是否应验: 静待时日.



# 20251218: 应该就近期逐步增持黄金, 还是等金融危机爆发时, 再买入黄金

- 本卦: 乾为天, 动爻初九, 变卦:天风姤
- 解卦: 近期不适合进一步增持. **关注突发性的市场震荡时买入, 重点关注公历5月.**
  - 本卦: 乾为天. 买卖: 不利买利卖. 
    - 乾为金, 本体之金!!!
    - 乾为天, 极盛之势, 暗示阶段性顶部? 至少不适合买入.
    - 问时间, 乾卦可表四月
  - 动爻: 初九, 潜龙勿用. 只可株守, 未可骤动. 
    - 逐步增持都是不适合的!
    - 应爻为四, 或跃在渊. 亦指向四月后.
  - 变卦: 天风姤
    - 不期而遇(时), 问的是买入黄金实体, 暗含乱世准备. 故综合判断下来, 直指交易时机的可能性较大. 
    - 姤卦主“遭遇”。这暗示适合买入的时机并非来自长期的阴跌，而是来自突发性的市场震荡.

# 20251231: 2026年, 金价是否有机会低于200澳元一克买入?

- 略筮法: 上卦3离, 下卦4震, 动爻1
- 本卦: 火雷噬磕, 动爻初九, 变卦:火地晋
  - 离卦可表金价.
- 解卦: **即便有机会也很难把握. 重点是公历5月金价有阻碍, 是买入之机, 后一路上扬.**
  - 本卦: 火雷噬磕.
    - 象征阻隔中有物，需以强力破除障碍
    - 理解为价格有阻碍. 两种理解:
      - 无法低于200澳元买入. (即便能到, 也可能转瞬即逝, 很难真正买入)
      - 金价会震荡为主,
  - 动爻: 初九. “履校灭趾”: 脚戴刑具, 伤及脚趾
    - 此惩为小惩大诫，可止其妄动，故“无咎”
      - 近期不要妄动.
    - 此象常示初期受制、价格受压、行动受限
      - 指行动受限, 也就是即便有机会, 也难以把握.
      - 指价格初期受压, 震荡市, 但跌幅有限. 
    - 应爻九四: 为噬磕的梗阻之物, 未光.
      - 四爻可应四月. 有梗阻, 导致无法扬升. 破后即再无阻力.
  - 变卦: 火地晋. 日出地上之象，象征光明渐盛、地位晋升、资源丰沛
    - 晋卦多主价格回升、价值显现. 最终单边上涨.
- 是否应验: 静待时日
