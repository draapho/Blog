# 爻的基础知识

- **阳爻(九)**: 主动、刚健、进取
- **阴爻(六)**: 主静、柔顺、守成
- **当位**, 或**得位**: 阳爻在奇爻, 阴爻在偶爻. 称之为当位.
  - 水火既济. 初九, 九三, 九五. 六二, 六四, 上六全部为当位.
  - 一般认为, 当位符合阴阳平衡, 象征吉利, 合理.  需记得物极必反.
- **不当位**, 或**失位**: 与当位相反, 阴爻在偶爻, 阳爻在奇爻
  - 火水未济. 初六, 六三, 六五. 九二, 九四, 上九全部为不当位.
  - 一般认为, 违背规律, 不合适宜, 不吉利或凶. 需记得物极必反.
  ![当位和不当位](https://draapho.github.io/images/2527/dang.jpg)
- **居中**: 二,五爻为"中", 所谓居中, 特指二爻或五爻.
  - 二爻与五爻是**全卦的枢纽**
    - **二爻**：内卦(下卦)之中, 代表**内在根基、德行、家庭、执行层**
    - **五爻**：外卦(上卦)之中, 代表**外在领导、趋势、天命、决策层**
    - **动爻居中**. 动在二，事关内修；动在五，事关外势。
  - 《彖传》《象传》凡论卦之吉凶, 十有七八聚焦二、五爻
- **中正**. 即爻即"居中", 又"当位". 称之为中正.
  - 六二,九五,为中正, 是“德位相配”之象, 为大吉之基
  - 可用内柔(六二)外刚(九五)为吉, 助记.
  - **二五居中, 中正为宝; 动在中位, 事系枢要**
  ![中正](https://draapho.github.io/images/2527/zhen.jpg)
- 爻位关系: **应**, **比**, 承, 乘. 
  - 见后续详解.




# 爻位

- 六爻不是静态符号，而是：
  - **时间之流**（从初到上）
  - **空间之序**（从地到天）
  - **人事之阶**（从民到君）
  - **心性之镜**（从隐到显）



## 三才与爻位

| 爻位      | 三才   | 象征                   | 主观能动性                          |
| --------- | ------ | ---------------------- | ----------------------------------- |
| 五爻,上爻 | 天(时) | 趋势、大势、共业、周期 | ❌ 大势所趋,共业所成. 可知可待.      |
| 三爻,四爻 | 人(德) | 心态、德行、智慧、行动 | ✅ 可修可为. 修心以应天, 聚力以用地. |
| 初爻,二爻 | 地(位) | 环境、资源、条件、局限 | ⚠️ 资源所依,境遇所限. 可避可转.      |

- 《周易》不会明确“何人何时何地做何事为吉”, 因为三才中的"人"始终是有主观能动性的, 是有机会去转换吉凶和观念的. "不三不四"这一词正是来自于三才的"人"对应于三爻四爻.
- 通常占卜的结果会暗含三才的所有信息, 这需要一定的解卦能力. 解卦时, 往往会以天时为背景, 强调人为的主观能动性, 即这种情况发生的话, 人怎么自处(心态和行动), 来保持人和, 或待天时转变, 或主动获取地利.
- 初学时, 解卦能力有限, 也可以拆分问题. 先问天时, 然后再断地利人和之事. 


## 爻位的特性

| 爻位     | 时序象征     | 人事象征        | 特性         |
| -------- | ------------ | -------------------- | ------------ |
| 上爻 | 事之极,穷尽. 结局,转向. | 退场, 过亢, 物极必反 | 亢, 悔, 当止 |
| 五爻 | 事之盛,鼎盛. 鼎盛,高峰. | 君位, 天命, 大德   | 尊, 中, 吉 |
| 四爻 | 事之变,近君. 变化,调整. | 外交, 高位, 近权   | 疑, 惧, 宜柔 |
| 三爻 | 事之进,躁动. 两难,抉择. | 进退两难, 多忧多惧 | 危, 慎, 需惕 |
| 二爻 | 事之成,显现. 明朗,守成. | 家庭, 内政, 德之基 | 中正, 安位  |
| 初爻 | 事之始,萌芽. 潜伏,初现. | 基层, 民间, 潜龙勿用 | 微, 隐, 可避 |

# 爻位关系

## 应

- **应**, 指上下两卦三爻的对应关系. 即初爻与四爻，二爻与五爻，三爻与上爻之间的呼应关系
- **相应**: 初与四、二与五、三与上，若一阴一阳，则“相应”. 
  - 多表内外通达、上下协力. 但需灵活判断
- **无应**或**敌应**: 初与四、二与五、三与上，若同阴同阳，则“无应”. 
  -  多表对立, 隔阂, 失助.
  - 两阳主亢，两阴主弱

![应](https://draapho.github.io/images/2527/yin.jpg)



## 比

- **比**: 指相邻两爻之间的关系. 相邻一阴一阳认为和谐, 多为吉; 同阴或同阳认为冲突, 多为凶. 
- 多用下邻爻关系. 主爻为上. 但需要灵活变通.
- 比有如下四种关系, 即:
  - **阳比阳**：刚进过甚（如《夬》九三“壮于頄”）
  - **阴比阴**：柔弱无援（如《剥》六二“剥床以辨”）
  - 阴阳相比：或助或阻，需看主爻
    - **阴下阳上**: 即**阴承阳**, 一般为助为吉.
    - **阳下阴上**: 即**阳乘阴**, 一般为阻为凶.
    - **例外**：若阴爻居尊(如六五), 乘刚可为“以柔制刚”之吉(如《谦》六五)
  - 可用内柔(阴下)外刚(阳上)为吉, 同性相斥助记.

![比](https://draapho.github.io/images/2527/bi.jpg)



## 承乘

- 个人观点: 承乘属于**比**的特例, 加上发音一样, 没有必要特别记忆.
- **承**: 承载, 奉持之意. 下承上. 一般用于阴下阳上的情况, 即阴承阳, 为吉. 
- **乘**: 压制, 凌驾之意. 上乘下. 一般用于阴上阳下的情况, 即阴乘阳, 为凶.
- 爻的相邻关系总结:
  - 比: 同阴, 同阳, 阴下阳上, 阳下阴上
    - 承: 仅表示下承上, 一般仅用于阴下阳上, 阴承阳的表述, 为吉
    - 乘: 仅表示上乘下, 一般仅用于阴上阳下, 阴乘阳的表述, 为凶



# 解爻要点

- **以动爻为中心**. 分析动爻相关的应、比
- **以卦主为中心.**
  - “卦主” = 主导全卦意义的核心爻. 常为唯一阴阳之爻, 即一爻为主, 众爻从之.
- **以问题为导向**
  - 譬如问“领导是否支持我?” , 则重点看五爻(领导)与二爻(我)是否相应；
  - 譬如问“基层是否稳定?” , 则重点看初、二爻关系。
- **确定中心后, 判断应、比的关系**
