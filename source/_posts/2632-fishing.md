# 基础需求

澳洲湖钓和有jetty的海岸边, 低成本休闲垂钓. 
目标暂定, 投放的虹鳟鱼, 和常见的鱿鱼.
其它鱼种暂时随缘, 有经验和兴趣后再研究.
最好有两根能互用又互补的鱼竿, 便于家人朋友分享参与, 一起钓鱼.



# 装备和鱼饵

装备:
- **6ft/1.8m钓竿 + 2500 纺车轮, 精细钓**. 公园虹鳟、溪流 Redfin、码头近身抽鱿鱼、小孩参与。
- **9ft/2.7m钓竿 + 3000 纺车轮, 全能钓**. 码头守真鲷(Snapper)、湖边远投沉底、应对稍强的海风。

鱼钩类别:
- **尺寸推荐:**
  - 钓投放的虹鳟鱼, 推荐7mm左右的钩子, 大致对应澳洲8号钩. 
  - 淡水鱼通用的话, 可以考虑8.5mm左右的钩子, 对应澳洲6号钩.
- **Baitholder Hook (带刺钩)：** 钩柄上有两个小倒刺。
  - 适合挂蚯蚓或整个小虾。倒刺能防止蚯蚓缩成一团或滑落。
- **Suicide / Octopus Hook (章鱼钩)：** 钩尖稍向内弯。
  - 适合挂玉米,虾肉块,鱿鱼条。
- **Circle Hook (圆型钩)：**
  - 适合挂powerbait面团, 让小钩完全埋入饵中, 可以漂浮起来(Powerbait 有浮力).

铅坠（Sinkers）. 
- 选择的原则是：**在能沉到底并定住的前提下，越轻越好。**
- **淡水湖**: 建议 1/4 oz (7g) 到 1/2 oz (14g)
- **海边码头**: 建议 1 oz (28g) 到 2 oz (56g)

饵钓:
- 红蚯蚓. 淡水通用饵 
- **罐头甜玉米**, 白面包团. 最便宜, 易获取. 通用饵
- PowerBait, 针对投放的虹鳟鱼, 比较贵.
- 海水钓, 可用生虾肉, 鱿鱼条, **蒜腌制的鸡块**

Lure钓:
- Tassie Devil (塔斯马尼亚恶魔): 虹鳟鱼, 推荐使用13.5g左右的.
- Spinner (亮片/旋转片): 虹鳟鱼
- Soft Plastics (软虫): 通用, 上手困难, 暂不考虑
- 木虾: 针对鱿鱼.



# 钓组及用法

主线末端, 两种选择:
- Surgeon's Loop 手术圈. 便于穿过鱼竿最小的孔.
- 八字环. 稳定方便, 但难以穿过鱼竿最小的孔.



针对虹鳟鱼饵钓:
- 主线Loop + 别针 → 0.2mm线(1米) → 7g子弹铅(可滑动) → 挡珠 → 8字环 → 0.2mm子线(60cm) → #8饵钩.  PowerBait 要揉实，完全包裹住鱼钩。
- 如果用玉米粒, 需要在钩子上前加一个5mm左右的泡沫浮珠. 对于穿孔浮珠, 在钩子前2-3cm处穿泡沫浮珠, 用太空豆挡住. 浮珠太大的话, 可以手工切割. 或者在钩子上串成玉米粒+浮珠+玉米粒. 需要进行浮力测试, 确保玉米粒能在水中漂浮起来.
- **用法：** 抛投到中远程，收紧余线，让竿尖稍微带一点张力。**PowerBait 会带着钩子漂浮在离底 60cm 的高度**（那是鱼嘴经过的高度）。可用抛掷若干玉米粒在钩子附近, 进行打窝.
- **特点：** 适合中午太阳大、鱼沉底不爱动的时候。



针对虹鳟鱼Lure钓:
- 主线Loop → 别针 + 0.2mm子线(60cm) → Devil鱼身(可滑动) → 挡珠 → #8大孔单钩
- **用法：** 抛投出去后，默数 3-5 秒让它下沉，然后**匀速收线**, 在收线过程中，偶尔**轻轻抽动竿尖（Twitch）**，模拟受伤小鱼的瞬间停顿。如果感觉没鱼，下次收线时稍微快一点或慢一点，寻找鱼所在的水层。 
- **特点：** 适合清晨、傍晚鱼群活跃时，大面积搜索湖区。 



墨尔本钓鱿鱼
- 主线Loop + 别针 → 0.2mm线(1.5米) →  别针(要不影响木虾活动, 如Egi Snap) → 3号木虾
- **用法**: 
  - 抛投后不要动，让木虾以45度角自然下沉。
  - 默数秒数（3号木虾3-4秒下沉1米）。根据水深，让木虾接近底部但不要挂草(留有余量, 可统一按3s/m算)。
  - **抽动 (Twitch/Jerk)：** 猛地向上快速抽动鱼竿 2-3 下。让木虾向上"之"字形串动.
  - **停顿 (The Drop)：** 停止抽动，收回余线，保持微绷，让木虾再次自然下沉。
  - **大多数鱿鱼都是在停顿这个阶段咬钩的, 重点关注鱼线鱼竿的手感.**
  - **中鱼后，不要猛抽鱼竿**（会撕断触手），而是**均匀、持续地收线**。保持压力，慢慢带出水面。如有必要, 使用网兜

  

通用软虫Lure钓: 
- 暂时不考虑.
- 全能型Lure钓, 但上手难度高, 不容易判断鱼讯。
- 需要练习“跳底”或“抽停”, 新手容易挂底.



# 常见目标鱼及钓法选择

| **鱼种**                         | **常见地点**           | **最佳钓法**           | **推荐杆子** |
| -------------------------------- | ---------------------- | ---------------------- | ------------ |
| **Rainbow Trout (虹鳟)**         | 公园人工湖             | 亮片、PowerBait 浮标钓 | 6ft          |
| **Redfin (赤鲈)**                | 大多数淡水湖、河       | 软虫、旋转片、蚯蚓     | 6ft          |
| **Flathead (牛鳅)**              | 海边 Jetty、沙底河口   | 软虫跳底、生饵底钓     | 8-9ft        |
| **Australian Salmon (澳洲鲑鱼)** | 海边 Jetty、河口       | 金属亮片快抽、生饵     | 8-9ft        |
| **Squid (鱿鱼)**                 | 墨尔本湾内多水草的码头 | 2.5-3.0 号木虾 (Egi)   | 6ft 或 8ft   |



Paternoster(串钩) 和 滑铅 (Running Sinker) 的目标鱼种

| **场景**         | **Paternoster (串钩)**                  | **Running Sinker (滑铅)**                     |
| ---------------- | --------------------------------------- | --------------------------------------------- |
| **底部**         | **水草丛生、乱石堆** (钩子悬空)         | **纯沙底、泥底** (饵贴底)                     |
| **目标鱼**       | **比较凶猛的鱼** (攻击性咬饵)           | **警觉性高的鱼** (尝试性吃饵, 无阻力感)       |
| **操作**         | **省心** (扔出去绷紧线即可)             | **需要观察** (鱼咬钩时要稍微送线)             |
| **墨尔本代表鱼** | **Salmon、Flathead、Trevally、Pinkies** | **Whiting (沙鮻)、Bream (黑鲷)、Trout(虹鳟)** |
