# 六爻占卜简介

六爻占卜源远流长，其理论体系可追溯至西汉时期的易学大师**京房**（公元前77-前37年）。京房在《京房易传》中系统阐述了”纳甲筮法”，将天干地支、五行生克等概念融入周易卦象之中，奠定了六爻占卜的基础理论框架。

《京房易传》云：”分六十四卦，配三百八十四爻，序一万一千五百二十策，定天地万物之情状。”这表明六爻体系通过卦爻的复杂组合，构建了一个能够模拟天地万物变化的象征系统。

历经两千余年的发展，六爻占卜在明清时期达到成熟。野鹤老人的《增删卜易》和王洪绪的《卜筮正宗》成为六爻学的经典著作，完善了用神、原神、忌神等核心概念，使六爻占卜更加系统化和实用化。



# 起卦方法详解

## 铜钱摇卦法
这是六爻占卜, 最传统的标准方法

- 准备工作与注意事项
    - 准备三枚大小、厚薄一致的铜钱（现代可用相同硬币代替）
    - 选择安静、整洁的环境
    - 心中默念所问之事，保持专注
    - 一事一卦，不重复占问同一问题
- 具体操作步骤
    1. 净手、静坐片刻，将问题明确思考清楚
    2. 将三枚铜钱合于掌心，轻微摇动
    3. 心中默念：”假尔泰筮有常，某（自己姓名）今以某事，未知可否，爰质所疑于神之灵，吉凶得失，悔吝忧虞，惟尔有神，尚明告之。”
    4. 将铜钱抛落在平整的桌面上
    5. 记录铜钱正反面情况：
       - 两正一反为**少阳**（记作 `一`）
       - 两反一正为**少阴**（记作 `--`）
       - 三反为**老阳**（记作 `一 ○`）
       - 三正为**老阴**（记作 `-- ×`）
    6. 重复以上步骤6次，自下而上记录六个爻
- 取得卦象
	- 自下而上记录六个爻, 就是本卦
	- 老阳老阴后面有 `○` 或 `×` , 就是动爻.
	- 动爻阴变阳, 阳变阴后, 就获得了变卦.

卜卦完成后, 需要一个纳甲的过程.

纳甲排盘的原理参考博客此文: [八宫卦和六亲世应图](https://draapho.github.io/2026/03/18/2621-bagonggua/)

- 核心原因: 心诚为上.
- 对于大多数人来说, 有一定的仪式感, 才可谓心诚.
- 虽不致于要求沐浴更衣, 但手取实物, 花几分钟闭息宁心, 卜得本卦还是必要的.




# 六爻占卜的适用范围

根据《增删卜易》，六爻占卜适用于：
- 个人决策：婚姻、事业、财运、健康等
- 时机选择：出行、签约、动土等吉日选择
- 事态预测：天气、考试、诉讼结果等

但不适用于：
- 国家大事（需用太乙、奇门等高级术数）
- 戏谑不敬之事
- 违背伦理道德之事



个人观点:
- 对于长远的(如一生之运), 宏大的(如国家大事), 也不是不行.
  - 一切事物为无中生有, 一生二生三, 螺旋成长而出.
  - 所占之事的大小, 也只是相对而言. 
  - 犹如 <红楼梦>, 将国家兴亡之事, 缩小到了大观园的家族尺度.
  - 但会完全违法六爻占卜中, 认为日月>动爻>静爻>卦辞爻辞的重要性排序.
- 因为卦已经在那里了, 只是怎么解卦的方法问题.
- 传统上, 六爻解卦以月建日辰为分析要点, 基本不看卦辞, 爻辞, 才有此说.
- 占卜之后, 解卦是可以根据情况选用解卦流派, 甚至杂糅而相互参考的.
- 当然这就对解卦者要求更高了. 需要针对具体问题, 或经验, 或灵感, 来解卦. 



# 解卦需要的核心概念

六爻占卜的核心是爻位和时间对应的十二地支, 十二地支有六冲六合, 以及对应到五行后的生克关系. 
其所有的概念都是确定本位后, 基于十二地支的冲合生克关系而来的.

## 爻神的定义
### 用神

根据问的事情，确定卦中哪个爻代表这件事。
**取对用神是六爻占卜的解卦中, 最为首要的事情。**
如果有多个相同的六亲爻, 即**用神多现**, 动爻优先, 次选持世或近世, 尽量避开空亡月破的爻.

| 问测事项                         | 取用神（六亲） |              逻辑              |
| :------------------------------- | :------------: | :----------------------------: |
| **父母、房产、文书、交通、天气** |   **父母爻**   | 父母为生(**支撑,庇护**)我之物  |
| **官职、疾病、工作、丈夫、盗贼** |   **官鬼爻**   | 官鬼为克(**管束,克制**)我之物  |
| **子孙、晚辈、宠物、解忧、医治** |   **子孙爻**   | 子孙为我生(**拥有,抚养**)之物  |
| **妻子、钱财、货物、饮食、工具** |   **妻财爻**   | 妻财为我克(**欲望,追求**)之物  |
| **兄弟、朋友、竞争、破财**       |   **兄弟爻**   | 兄弟为比和(**分享、竞争**)之物 |



### 原神、忌神、仇神、耗护神

确定用神之后, 原神、忌神、仇神都是固定下来的, 可以直接对应到某个六亲爻.
用神、原神、忌神、仇神、耗护神都是临时变量, 确定用神六亲后, 其它神的六亲就固定下来了.

| 概念       | 定义                     | 比喻           | 作用                                     |
| :--------- | :----------------------- | -------------- | :--------------------------------------- |
| **用神**   | 占卜的核心目标           | 我军           | 确定吉凶的承受主体。                     |
| **原神**   | **生用神**之爻           | 粮草, 救兵     | 判断用神是否有源、有救、有后劲。         |
| **忌神**   | **克用神**之爻           | 敌军           | 判断阻力、风险、破坏力的来源。           |
| **仇神**   | **用神克**之爻. 必生忌神 | 敌军粮草       | 当忌神旺相时，仇神会加剧危害。初期可略。 |
| **耗护神** | **用神生**之爻           | 分兵, 培养新军 | 会耗泄用神的气，但可克忌神.              |



爻神和六亲关系对应表

|   用神   | 原神(生用神) | 忌神(克用神) | (用神克)仇神(生忌神) | (用神生)耗护神 |
| :------: | :----------: | :----------: | :------------------: | :------------: |
| **父母** |     官鬼     |     妻财     |         子孙         |      兄弟      |
| **子孙** |     兄弟     |     父母     |         官鬼         |      妻财      |
| **官鬼** |     妻财     |     子孙     |         兄弟         |      父母      |
| **妻财** |     子孙     |     兄弟     |         父母         |      官鬼      |
| **兄弟** |     父母     |     官鬼     |         妻财         |      子孙      |



### 飞神和伏神

*初学暂缓.*

另有一种情况, 所得的卦, 会缺某个六亲, 这样对应的爻神就会空缺. 这种情况, 需要找**伏神**代替.

- **伏神**: 找到卦所属的宫, 宫主卦必有需要的六亲对应的爻, 将此爻定为**伏神**即可.
- **飞神**: 确定伏神所在爻之后, 原卦该爻的六亲, 就叫**飞神**

打个比方: 求财得"天风姤"卦. "天风姤"六亲没有妻财爻.
- 可知"天风姤"属于乾宫, 找到宫主"乾为天"的六亲世应图.
- "乾为天"的妻财爻, 在二爻, 为妻财寅木. 将其挂在本卦二爻后面.
- 回到本卦"天风姤", 即没有**用神**. **伏神**为妻财寅木;
- **飞神**为"天风姤"二爻的六亲: 亥水子孙
- 断卦时，重点还是本卦的6个爻. 伏神只在分析"财"的时候才调出来参考.

伏神的三种状态：

| 状态         | 条件                                                         | 结果                      |
| :----------- | :----------------------------------------------------------- | :------------------------ |
| **伏不得出** | **飞神克伏神** 或 **伏神休囚死** 或 **无日月生扶**           | ❌ **完全不可用**          |
| **伏而待出** | **飞神空亡** 或 **飞神月破** (飞神 被月建冲) **或** 3. **日月生扶** (日/月 生 伏神，或 五行 同 于 伏神) | ⏳ **未来可用** (需待应期) |
| **伏而得出** | 1. **飞神生伏神** (飞神五行 生 伏神五行) 2. **伏神旺相** (伏神在月建为 旺 或 相) 3. **日月生扶** (日/月 生 伏神，或 五行 同 于 伏神) | ✅ **当前可用**            |



## 世应爻和动静爻

世应的含义:
- **世爻：** 代表**占卜者自己**。
- **应爻：** 代表**对方、环境、所测之事**。

世应的关系:
- 世应相生：表示双方和谐，事情容易成功
- 世应相克：表示存在矛盾，需要化解
  - 世克应：自己主动，有利于自己
  - 应克世：对方主动，对自己不利
- **世应的应克关系只是参考之一. 需要看用神.**

动静变化的含义:
- 动爻：有变化，主动，力量强. 表示事物运动状态. 
- 静爻：无变化，被动，力量弱. 表示事物稳定状态
- 变爻：动爻变出的爻，代表事情的发展趋势.
- 动爻为重，静爻为轻，动爻在断卦中具有重要作用。
- 动爻生克静爻：动爻对静爻有影响

### 十大六冲卦, 八大六合卦

**十大六冲卦**（固定名单）
- 八个宫主卦：乾、兑、离、震、巽、坎、艮、坤
- 两个天雷卦: 天雷无妄、雷天大壮
- **特征**：主散、快、冲突、变动。

**八大六合卦**（固定名单）
- 两个天地卦: 天地泰、地天否
- 两个地雷卦: 地雷复、雷地豫
- 两个泽水卦: 水泽节、泽水困
- 两个山火卦: 山火贲、火山旅
- **特征**：主合、慢、稳定、羁绊。

| **问出行**      |    **忌合**    |           合代表走不了，被事情耽搁，行程延误。           |




## 月建日辰

日月时间, 取十二地支对应的五行即可. 对应关系见"五行生克"中的介绍.
月建日辰的基础概念.
- **月柱**为一月的天干地支.
- **月建**为一月的地支部分.  六爻主要使用地支, 因而称月建.
- 月建是按节气更换的, 而不是按农历初一更换.
- **日柱**为一日的天干地支. 
- **日辰** 为一日的地支部分. 六爻主要使用地支, 因而称日辰.

**月建日辰的作用力**：
- 月建：当月地支，管一月之权
- 日辰：当日地支，管一日之权

**日月对爻的影响：**
- 月建生扶则旺，月建克制则衰. 有 **旺相休囚死** 之判断
- 日辰生扶则增力，日辰克制则减力

### 日辰的空亡

**空亡, 也叫旬空.** 
中国天干地支纪年法中, 有十天干, 十二地支. 阴配阴, 阳配阳, 依次相配. 故而一轮为60.
旬表十, 这里指十天. 十个天干只能配到十个地支, 这轮空的两个地支, 就叫旬空, 即为空亡.
如甲子旬, 戌亥空亡. 甲戌旬, 申酉空亡.

空亡通俗的理解：**空亡 = 暂时掉线、能量不在线、心里没底**。分真空/假空

### 应期

对于空亡的爻, 有应期的概念, 即何时能回归岗位起作用, 是未来作用时间的重要判断依据
- 主要看假空爻的应期, 通常表示起作用的时间.
- 真空爻也有应期，真空爻的应期多主：**应而不实 / 应凶 / 彻底落空**

| 术语            | 触发条件                     | 通俗比喻                         | 适用场景                     | 举例（戌亥空）                    |
| --------------- | ---------------------------- | -------------------------------- | ---------------------------- | --------------------------------- |
| **填实**        | 空亡地支「值日/值月」        | 「请假的人今天到岗了」           | 最直接、最彻底的解空方式     | 戌爻空 → 逢戌日/戌月              |
| **出空**        | 过了当前旬（10天），进入新旬 | 「假期结束，自动返岗」           | 时间自然流转解除空亡         | 甲子旬戌亥空 → 进入甲戌旬自动出空 |
| **逢冲** (冲实) | 空亡爻逢对冲地支来冲         | 「被闹钟叫醒，被迫上岗」         | 需爻有气（假空），真空难冲起 | 戌爻空 → 逢辰日（辰戌冲）         |
| **逢合**        | 空亡爻逢六合/三合            | 「被队友拉起来一起干」           | 辅助解空，需结合卦象判断     | 戌爻空 → 逢卯日（卯戌合）         |
| **动而逢值/冲** | 动爻空亡 + 逢值日/逢冲日     | 「主动请假的人，一到时间就发力」 | 动爻应期更快，优先判断       | 戌爻发动且空 → 戌日或辰日应事     |



## 动爻和变爻的专用概念

### 伏吟(地支相同)

**伏吟**: 即动爻的地支和其变爻的地支相同. 必然五行比和(旺),
- **象征事物停滞、反复、内心纠结、进展艰难的状态。**
- 可理解为能量积蓄期, 事物发酵等待期.
- 除了地支伏吟, 还可以参考六兽伏吟(天干相关), 六亲伏吟.
**伏吟应期**: 地支的冲或合, 或值.
- 冲, 主动求变, 打破僵局.
- 合, 安抚缓解, 暂得安宁. 但非根本解决.
- 值, 渡过积蓄等待期, 最终落地. 
- 需要根据动爻变爻的能量, 确定是否能"冲"开. "值"的时候, 最终落地是成还是败.



### 反吟(地支相冲)

**反吟**:即动爻的地支和其变爻的地支相冲，形成"来回冲撞"之象。
- **象征事有反复、动荡、变化快、来回折腾、事有反复**
**反吟应期**: 地支的冲或合, 或值.
- 动爻值, 变爻冲, 加剧变动, 极限. 后或缓或破. 可理解为事物发展的一个极值.
- 动爻合, 波动暂时压制而稳定, 非根治, 只是缓和.
- 动爻冲, 变爻值, 事态最终落地.
- 需要根据动爻变爻的能量, 判断应期是否可用可信.



### 化合(地支相合)
**化合**: 即动爻的地支和其变爻的地支相合. 
- 主**牵绊, 合作, 稳定, 缓慢**
- 应期为: 动爻冲或变爻冲. 解开羁绊. 同样需要看能量情况



### 五行关系
动爻和变爻的五行关系如下. 其强弱结果主看动爻的"旺相休困死"
- 变生动, 变休, **动相, 回头生**, 事后得助, 加强结果. 气聚, 最强
- 动变比和, **动变两旺**. 根据地支变动分**化进化退**. 总体相助且平稳. 气和, 强
- 动生变, 变相, **动休, 化泄**. 动爻耗泄, 助力结果. 气泄, 平
- 动克变, 变死, **动困, 化克**, 动爻付出, 结果有限. 气克, 弱
- 变克动, 变困, **动死,回头克**, 事后有损, 结果反噬. 气死, 最弱



### 进神、退神

仅针对动爻与变爻同五行的情况下, 比较两者的地支. **地支循环, 小步前进为进神, 小步后退为退神. **

| 概念     | 定义                 | 比喻     | 作用               |
| :------- | :------------------- | -------- | :----------------- |
| **进神** | 动爻爻变后比原爻更旺 | 士气提高 | 事情向好的方向发展 |
| **退神** | 动爻爻变后比原爻更弱 | 士气降低 | 事情向坏的方向发展 |

## 地支的三合三刑六害

## 三合

三合局是由**三个地支**组合成一个**五行局**。有 4 组
成局条件严格, 只针对动爻+日月的情况.
一旦成型, 就是团队力量, 比六合六冲力量还大。
主: **聚力成势**, **强力有效**, **长久稳定**. 常用来断**长远应期**.

| 三合局     | 合成五行 | 比喻               |
| :--------- | :------- | :----------------- |
| **申子辰** | **水局** | 水势浩大（如洪水） |
| **亥卯未** | **木局** | 木气成林（如森林） |
| **寅午戌** | **火局** | 火势燎原（如火山） |
| **巳酉丑** | **金局** | 金气成器（如兵阵） |



| 情况                    | 是否成局   | 说明                                       |
| :---------------------- | :--------- | :----------------------------------------- |
| **卦中三爻皆动**        | ✅ **成局** | 三个动爻正好凑齐三合（如申子辰全动）       |
| **两爻动 + 占卜日或月** | ✅ **成局** | 卦中两动爻 + 日/月补齐第三个地支（最常见） |
| **一爻动 + 占卜日月**   | ✅ **成局** | 卦中一动爻 + 日/月补齐另两个地支           |
| **仅缺一个五行**        | ⏳ **待用** | 叫“**虚一待用**”，等未来时间补齐那个字     |



三合局的"破局"情况供参考. 三合局成立后，也可能被"打破".
注意：**三合局一旦成立，力量很强，不易被破**，除非冲的是"中神"（子/卯/午/酉）。

| 破局方式 | 说明                        | 例子                                |
| :------- | :-------------------------- | :---------------------------------- |
| **冲局** | 日/月冲三合局中的任何一个爻 | 申子辰水局，逢午日冲子 → 局破       |
| **空亡** | 三合局中任一爻旬空          | 申子辰，子水旬空 → 局虚，待出空     |
| **月破** | 三合局中任一爻被月建冲      | 申子辰，子水在午月 → 子月破，局不稳 |

|              | 假无气          | 表面休囚，但有暗动/伏神/合局           | ⚠️ 隐性能量       | 待时而发，出空/逢冲可用  |
| **假空应期** | 填实            | 空亡地支, 值日/值月                    | ✅ 能量回归       | 最直接的解空方式         |
| (真空可看)   | 出空            | 过当前旬(10天)，进入新旬               | ✅ 自动解除       | 时间流转自然解空         |
| (但主应凶)   | 逢冲(冲实)      | 空亡爻 逢对冲地支                      | ⚠️ 需爻有气       | 冲开空亡，激发能量       |
| (问灾反吉)   | 逢合            | 空亡爻 逢六合/三合                     | ⚠️ 辅助解空       | 合中带解，需看卦象       |
|              | 动而应期        | 动爻空 + 逢值/逢冲                     | 🚀 应期更快       | 动爻优先，发力迅速       |

## 库墓

*高阶应用, 初学暂缓.* 其理论基础基于长生十二宫. 



墓库之法，理法定旺衰吉凶，象法断具体事象

- 爻只入**高层次**之墓库（日/月 > 变爻 > 动爻），不入同层次墓
- 低层次的爻, 只有匹配了墓库的五行要求, 才能入库入墓. 
  - 为土的爻, 固定按"土随水行"标准，统一入辰库。若遇"土入戌"之说，知为流派不同即可。
  - **戌土的爻, 优先考虑"辰戌冲", 冲力大于墓库力.**
  - 譬如丑墓, 只能接受五行为金的爻入墓. 其它五行无法入墓.
- 入墓库期间，爻**暂时失去生克权**，不能生克他爻，他爻也不能生克它

| 墓库地支 | 可入墓库的爻五行 | 旺爻入"库"               | 衰爻入"墓"                 | 特殊象意               |
| -------- | ---------------- | ------------------------ | -------------------------- | ---------------------- |
| **辰**   | 水, 土. (戌辰冲) | 水库：钱财积蓄、资源待用 | 水墓：被困、昏迷、事停滞   | 地户、东南、基层.      |
| **戌**   | 火               | 火库：能量储备、贵人暗助 | 火墓：熄灭、希望落空、病重 | 天门、西北、远方、升迁 |
| **丑**   | 金               | 金库：财富收藏、技术沉淀 | 金墓：才华埋没、器械损坏   | 冰库、寒冬、隐忍       |
| **未**   | 木               | 木库：计划酝酿、生机潜伏 | 木墓：生机断绝、计划夭折   | 花园、田园、饮食       |



按六亲入库入墓取象

| 六亲     | 卦吉爻旺 → 入库                   | 卦凶爻衰 → 入墓               |
| -------- | --------------------------------- | ----------------------------- |
| **妻财** | 钱财存银行✅、投资待涨、支票在手   | 钱财丢失❌、资金冻结、债务缠身 |
| **子孙** | 学生在校学习✅、医生值班、药品储备 | 孩童走失❌、医生失职、药品短缺 |
| **兄弟** | 朋友帮忙✅、团队集结、手足团聚     | 朋友失联❌、手足受伤、被人拖累 |
| **官鬼** | 工作稳定✅、职位在编、官司有记录   | 疾病潜伏❌、小人暗算、官非缠身 |
| **父母** | 文书归档✅、房产在手、长辈安康     | 证件丢失❌、房屋破损、长辈病困 |



**入库入墓**的三种形式(爻如何入库墓)

- 入日/月墓库：符合五行的爻逢日辰或月建为墓库
  - 例：申爻逢丑日/丑月 → 申金入丑墓
- 入动爻墓库：静爻可入卦中动爻之墓
  - 例：子水静爻，卦中辰土动 → 子水入辰墓
- 入变爻墓库：动爻可入变爻之墓. 
  - 表示是爻自己的行为, 常表示结局和趋势.
  - 这种情况非常重要, 有固定名称 **化墓化库**.
  - 例：午火动化戌土 → 午火入戌墓



**出库出墓**的三种形式(爻如何出库墓)

- 出日/月墓库：待未来之日/月冲开墓库(日/月)，或值入墓库之爻
  - 因日/月力量大，卦爻不能冲日/月，只能**等时间**
  - 例：申爻逢丑日入墓 → 待**未日冲丑**（冲开墓），或**申日值爻**（爻旺出墓）
- 出动爻墓库：日/月冲墓库(动爻)，或值入墓库之爻(静爻)
  - 墓是卦中动爻，层次较低，可被日/月冲开
- 出变爻墓库：日/月冲墓库(变爻)，或值入墓库之爻(动爻)
  - 表示结局可解，待时而动，应期多在此时
  - 这种情况对应“化墓化库”，解墓即解局
- 另有一种暗动出墓的情况, 供了解参考.
  - 当入墓库之爻本为"旺相", 未来日月地支冲了此爻, 可认为推醒了此爻. 即**暗动出墓**
  - 但如果此入库墓之爻为"休囚死", 未来日月冲了此爻，则为凶，难出墓.



对于墓库, 自己的理解和总结: 

- 地支为 "辰戌丑未" 四个五行为土的地支, 可为墓或库.
- 墓库, 可以理解为一个暂存, 一个临时箱子, git里的staging. 
- 低层次的爻, 可以入高层次的墓库. (爻可入日月墓库, 静爻可入动爻墓库, 动爻可入变爻墓库)
- 入了墓库的爻, 相当于被关了起来. 入墓后失去主动五行生克权. 
- 作为墓库的高层次的日/月或爻, 保留原功能, 依旧可以参与判断.
- 出库出墓, 主要看未来的日月时间, 是否能冲开墓库, 或值入墓之爻.
- 由解卦者灵活判断是否要进行入库入墓的操作. 因而比较难用.
